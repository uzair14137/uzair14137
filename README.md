I'm a highly self-motivated and goal-oriented **Data Engineering** professional based in **Islamabad, Pakistan**, passionate about designing and building scalable data solutions with real-time and batch processing capabilities.

- 🏗️ &nbsp;Currently building **Lakehouse architectures** with Apache Iceberg, Delta Lake & MinIO
- 🔄 &nbsp;Experienced in **Medallion Architecture** (Bronze → Silver → Gold) for robust data pipelines
- ☁️ &nbsp;Working with **AWS** and **Snowflake** for cloud-native data engineering
- 🚀 &nbsp;Skilled in **Apache Spark, Airflow, Kafka** for end-to-end data orchestration
- 📊 &nbsp;Focused on transforming raw data into actionable **business insights**
- 🌱 &nbsp;Always learning new technologies and best practices in the data space
